# 🐾 Pawshot for Windows

> **スクリーンショット × OCR × アノテーション** を一発で。Windows常駐型のキャプチャツール。

---

## ✨ 機能一覧

### 📸 キャプチャ
- **グローバルホットキー** `Ctrl + Shift + 6` でどこからでも即起動
- **範囲ドラッグ選択**でキャプチャ範囲を指定
- キャプチャ画像を**クリップボードに自動コピー**

### 🔤 OCR（文字認識）
- Windows 標準の `Windows.Media.Ocr` エンジンを使用
- キャプチャ直後にテキストを自動抽出
- 抽出したテキストを**クリップボードにも自動コピー**
- DPI スケーリング対応（高解像度モニター対応）

### 🖊️ アノテーション（注釈）ツール
キャプチャ後に表示される編集画面で、以下の操作が可能：

| ツール | 説明 |
|--------|------|
| ▶ 選択ツール | 図形・テキストを選択、移動、拡大縮小（マウスホイール） |
| □ 四角形 | 矩形の枠を描画 |
| ○ 円形 | 楕円・円を描画 |
| ↗ 矢印 | 矢印を描画 |
| T テキスト | テキストを配置（ダブルクリックで再編集） |
| ⬛ ぼかし | モザイク / ぼかし処理でマスク |

#### アノテーション詳細機能
- 🎨 **カラーパレット**: 赤・青・緑・黄・白・黒の6色
- ✏️ **線の太さ**: 細・普通・太の3段階
- 🔵 **選択枠表示**: 選択中のオブジェクトに青い点線の枠を表示
- 🗑️ **個別削除**: 選択状態で `Delete` または `Backspace` キー
- ↩ **元に戻す**: `Ctrl + Z`
- ♻️ **スケール**: `ScaleTransform` を使った本来の拡大縮小（太さ変更ではなく）

### 🖼️ フローティングプレビュー
キャプチャ・編集後、画面隅に小さなプレビューウィンドウを表示。
そこから**保存・コピーなど**のクイックアクションを実行可能。

---

## 🚀 使い方

1. アプリを起動するとシステムトレイに常駐します
2. `Ctrl + Shift + 6` を押す
3. キャプチャ範囲をドラッグで選択
4. アノテーションウィンドウで注釈を追加
5. 「完了」ボタンで画像をクリップボードに保存

---

## 🛠️ 動作環境

- **OS**: Windows 10 / 11
- **ランタイム**: .NET 8.0 以上
- **フレームワーク**: WPF (Windows Presentation Foundation)

---

## 📦 ビルド方法

```bash
# リポジトリをクローン
git clone https://github.com/jsmsk-ico26/Pawshot_Windows.git
cd Pawshot_Windows

# ビルド
dotnet build

# 実行
dotnet run
```
