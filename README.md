# 🐾 Pawshot for Windows

> **スクリーンショット × OCR × アノテーション** を一発で。Windows常駐型のキャプチャツール。

---

## ✨ 機能一覧

### 📸 キャプチャとモード切替
2つのショートカットを使い分けることで、用途に合わせたキャプチャが可能です。

- **`Ctrl + Shift + 6`：アノテーションモード**
  - 範囲選択後、編集ウィンドウ（矢印、文字入れ等）が開き、加工してから保存・コピーができます。
- **`Ctrl + Shift + 7`：OCR専用モード**
  - 範囲選択後、**即座にテキスト抽出**を行い、クリップボードにコピーします（編集画面はスキップ）。

### 🔤 OCR（文字認識）
- Windows 標準の `Windows.Media.Ocr` エンジンを使用。
- DPI スケーリング対応（高解像度モニターでも正確に座標を取得）。
- 抽出テキストは自動的にクリップボードへコピーされます。

### 🖊️ アノテーション（注釈）ツール
キャプチャ後に表示される編集画面で、以下の操作が可能：

| ツール | 説明 |
|--------|------|
| ▶ 選択ツール | 図形・テキストを選択、移動、拡大縮小（マウスホイール） |
| □ 四角形 | 矩形の枠を描画 |
| ○ 円形 | 楕円・円を描画 |
| ↗ 矢印 | 矢印を描画 |
| T テキスト | テキストを配置（ダブルクリックで再編集） |
| ⬛ ぼかし | モザイク / ぼかし処理で隠したい場所をマスク |

#### アノテーション詳細機能
- 🎨 **カラーパレット**: 赤・青・緑・黄・白・黒の6色。
- ✏️ **線の太さ**: 細・普通・太の3段階。
- 🔵 **選択枠表示**: 選択中のオブジェクトに青い点線の枠を表示し、操作対象を明確化。
- 🗑️ **個別削除**: 選択状態で `Delete` または `Backspace` キーで要素を削除。
- ↩ **元に戻す**: `Ctrl + Z` 対応。
- ♻️ **スケール**: 太さを変えるのではなく、図形そのものを拡大縮小（ScaleTransform）。

## 🚀 使い方と配布

### 実行方法 (EXE)
- `Pawshot_Windows.exe` を実行するだけで起動します。
- 自己完結型（Self-contained）でビルドされているため、別途 .NET ランタイムのインストールは不要です。
- 起動するとシステムトレイ（画面右下のアイコン群）に常駐します。

### 設定ファイル
- 初回起動時に同じフォルダに `config.json` が自動生成されます。
- 保存先の設定などを保持するために必要なファイルです。

### OCR が動作しない場合
日本語の OCR を利用するには、Windows の設定で「日本語の手書き文字認識」または「言語パック」がインストールされている必要があります。
- `設定` -> `時刻と言語` -> `言語と地域` -> `日本語` のオプションからインストール状況を確認してください。

## 📦 開発者向けビルド方法

```bash
# リポジトリをクローン
git clone https://github.com/jsmsk-ico26/Pawshot_Windows.git
cd Pawshot_Windows

# 実行
dotnet run

# EXE (単一ファイル) の生成
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ./publish
```
